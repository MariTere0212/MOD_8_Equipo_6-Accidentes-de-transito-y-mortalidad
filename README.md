# MOD_8_Equipo_6-Accidentes-de-transito-y-mortalidad
Análisis de la siniestralidad vial y sus factores asociados. El objetivo de este proyecto es explorar los datos abiertos de accidentes viales, visualizar sus tendencias y realizar un análisis descriptivo que permita comprender mejor dónde, cuándo y cómo ocurren los accidentes.
